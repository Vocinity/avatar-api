## Instalation
- Download: https://www.gyan.dev/ffmpeg/builds/ffmpeg-git-full.7z
- unzip to:  C:\ffmpeg
- System properties -> Advanced tab -> Enviroment Variables
- Select Path variable, then Edit button
- Add new entry to: C:\ffmpeg\bin
- click Ok
- click Ok
